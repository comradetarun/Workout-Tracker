# Workout Tracker

## Overview

Workout Tracker is a fitness tracking application developed as an academic project. The goal of the project was to help users monitor and manage their fitness activities.

## Features

* User registration and login
* Workout tracking
* Daily activity monitoring
* Calorie tracking
* Goal setting
* Progress monitoring
* Sleep tracking

## Technology Stack

* React.js
* Node.js
* Express.js
* MongoDB

## Database Collections

### Users

Stores user profile and login information.

### Workouts

Stores workout details such as workout type, duration, and calories burned.

### Activities

Stores daily activity records.

### Goals

Stores fitness goals and target achievements.
